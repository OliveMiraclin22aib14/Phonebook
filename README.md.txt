Phonebook Management System in C

A simple Phonebook Management System in C that uses file handling and structures to add, view, search, update, and delete contact details.

Features
- Add new contact
- View all contacts
- Search contact by name or number
- Update contact details
- Delete contact
- Data stored in a text file for persistence

How to Run
1. Compile `phbook.c` using a C compiler.
2. Run the generated executable.
